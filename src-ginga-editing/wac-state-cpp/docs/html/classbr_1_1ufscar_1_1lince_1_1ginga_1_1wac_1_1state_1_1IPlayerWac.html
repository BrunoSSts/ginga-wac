Detailed Description

Interface utilizada para se obter o PlayerState das classes Players do módulo Formatter.

Esta interface deve ser implementada pelas classes PlayerAdapter do módulo Formatter do Ginga para que seja possível se obter os estados correntes de cada player em execução.
